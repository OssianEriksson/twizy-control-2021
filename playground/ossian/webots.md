# Twizy simulation in webots

# Installing

1. Install [webots_ros](http://wiki.ros.org/webots_ros#Installation) and it's dependicies.

2. Setup the `WEBOTS_HOME` enviroment variable with
   ```sh
   echo export WEBOTS_HOME=/usr/local/webots >> ~/.bashrc
   source ~/.bashrc
   ```

3. Istall robot_localization:
   ```
   apt install ros-$ROS_DISTRO-robot-localization
   ```
